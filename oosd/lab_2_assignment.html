Some notation information

  • Angular brackets containing descriptive text are placeholders, used to describe required input or other literals. For example,
    • if an expected input string value is described as <name> then the actual value could be John or Mary
    • if a value is described as <date of birth in format dd/mm/yy> then the actual value might be 01/01/01

    The actual value should not include angular brackets.

  • If a command line window is shown to demonstrate requirements,
    • text printed by the programe is displayed in white
    • user input is displayed in red
    For example,
    Please enter an number: 42

General instructions

In the submitted code you must use only

  • the constructs covered in class so far
  • any additional string type methods you might need

Exercise 1: Cutting the grass

Write a Python program that does the following

  • reads in a garden length, having appropriately prompted the user
  • reads in a garden width, having appropriately prompted the user
  • reads in a house length, having appropriately prompted the user
  • reads in a house width, having appropriately prompted the user
  • assuming that the garden in fully covered in grass, except for the area where the house is, calculate how long it will take to mow the lawn at a speed of 0.5 square meters per second
  • tell the user how long it will take to cut the grass in their garden

Exercise 2: Name formatting

Write a Python program that does the following:

  • reads in a name and a surname, typed in with a space between them (<first name> <surname>)
  • processes the input and prints out the name in the format <surname with only first letter capitalised>, <first name initial as a capital>)
  • handles any combination of capitals and lower case letters in the input

Input and output examples (note that no other text is needed either with the input or output):

inputoutput
John DoeDoe, J
jane doeDoe, J
TOM BLACKBlack, T
eva GREENGreen, E

Exercise 3: Item reordering

Write a Python program that carries out the following interaction with the user (the user input, in red, is only a sample):

Please the list of 5 items to be rated, separated by spaces: bicycle scooter rollerblades rollerskates skateboard
----------------------------
Item ids:
1 bicycle
2 scooter
3 rollerblades
4 rollerskates
5 skateboard
----------------------------
Please enter the item ids in order of your preference for the items, separated by spaces: 1 3 4 2 5
Please enter the item ids in order of your friend's preference for the items, separated by spaces: 5 3 4 1 2
----------------------------
The items ordered by your preference: bicyle rollerblades rollerskates scooter skateboard
The items ordered by your friend's preference: skateboard rollerblades rollerskates bicycle scooter
----------------------------

Tips:

  • store item list and id preference lists in list variables
  • use the ids in the preference lists to access elements of the item list by index
  • be careful when using the ids: they start at 1, while Python list indices start at 0
