Questions on variable types - answers
  1. Types:
    • ID: not for analysis
    • width: the variable as a concept is continuous numeric on the ratio scale, but the data are discrete if whole numbers are allowed only (this seems to be the cse)
    • colour: nominal categorical
    • shape: nominal categorical
  2. If these data were used for some physics experiment, maybe, where the wavelength of the colour is relevant.
  3. Continuous numeric on the ratio scale
  4. The variable as a concept is continuous numeric, however, the data collected with a digital scales would be discrete. Ratio scale.
  5. Dichotomous (present/absent) or
    other nominal categorical (present/absent/late/excused) or
    discrete numeric (overall number of attendances) on the ratio scale.
  6. Continuous numeric on the ratio scale. In principle this applies both for height from sea level and from trailhead altitude, but it is in the second case that the ratio scale makes more practical sense.
