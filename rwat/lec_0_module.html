Labs
These are each worth 1%, which is given for a reasonable effort (not a complete and fully working submission). The purpose of the labs is learning.
Lectures - practical topics
These are covered in conceptual terms. You are expected to use MDN or other reliable reference material to find out details on how to code with the concepts. The main concepts and technologies are included as requirements in the labs, but otherwise different students may focus on different details. The purpose of this material is to expose you to as many different technologies so that you become familiar with the client-side development landscape and to aquire some practical skills. The CAs test if you have reached some level of skill.
Lectures - theory
As the word says, this is the theory behind the practice. This should provide conceptual understanding needed to answer exam questions.
Summative assessment: CAs
These assess the practical skills you have acquired through lab work
Summative assessment: exam
This assesses your conceptual understanding of client-side Web technologies
