Reactive programming

Prerequisites

For this lab you need to have node.js installed. If you have completed the BaaS lab, you will have installed it already.

Assignment

Your assignment is to implement a small application with the use of reactive programming. The requirements are as follows.

  • Use the RxJS library. Documentation for it can be found here. To install it, do the following.

    Create an npm project by running

    npm init
    in a directory you have created for the lab. Then run
    		  
npm install vite
npm install rxjs
    Vite is not a prerequisite for RxJS, but it will help with development. See the BaaS lab instructions for more details.

  • Create an application in a single HTML file and a single JavaScript file (also CSS if you wish) that detects groups of clicks (like a double-click but including any number of clicks greater than 1) on the page and displays the number of clicks in the group on the HTML page.
  • The main challenge in this lab is to develop an algorithm for detecting groups of clicks. You might start by defining the longest interval that counts as a 'within-group' interval and the shortest interval that counts as a 'before-group' or 'after-group' interval. You must also take into account the relative lengths of the 'within' and 'before/after' intervals.
  • Use RxJS to implement the application. Start by creating an observable with the use of the fromEvent function. Then find and use suitable operators for your implementation.
  • The HTML page should not have anything on it except the title and the element that displays the number of clicks
