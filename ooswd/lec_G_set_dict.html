Python built-in container types

There are four types of built-in container types in Python:

  • lists
  • tuples
  • sets
  • dictionaries

So far in the module we have used the first two. In this session we will look at sets and dictionaries.

  • The Python set container contains unique elements (like in maths). It can be used, among other things, for testing membership and identifying unique values in lists.
