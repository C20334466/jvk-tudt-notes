Problem-Based Learning Assignment: School Uniform Orders

  • Your assignment is to write a Python program that automates online ordering for PrimaryFashions, a company that sells primary school uniforms.
  • Except where indicated, your program may assume that user input is going to be valid (i.e. error handling should be implemented only if stated as an explicit requirement).
  • Where user input is required, suitable prompts must be provided, e.g. Please enter your name:
  • Where output is required, suitable descriptions must be provided, e.g. printing an order total might read Your order total is €100 (not just €100).
  • The program should:
    • print a title banner *********** PrimaryFashions Online Shop *********** when it starts
    • read in the user's name
    • read in the user's post code, check that it is exactly seven characters long and that it consists of alphanumeric characters only (the string method isalnum can be used for this), then if the post code is invalid read it in again, repeating the process until a valid code is obtained
    • modify the postcode so that any lower-case letters are converted to upper-case
    • read in the name of the first child for whom a uniform is being ordered (so that the name can be embroidered)
    • read in information about items that are being ordered for the child:
      • item types: jacket, jumper; one of these or both may be ordered for the child
      • item sizes: S (small), M (medium), L (large); one size should apply to both items for the same child
    • ask the user if they wish to enter information for another child, repeating the process for as many children as the user wishes
    • calculate the bill total if the prices are as follows:
      • small jumper: €25
      • small jacket: €40
      • any medium-sized item: 6% more than the same type of item in size S
      • any large-sized item: 10% more than the same type of item in size S
    • print out an order summary, quoting the customer's name, postcode and order total
    • print out the list of items (in any order), with lines in the form: 
      
item_type   item_size   name_to_be_embroidered
      e.g. 
      
jumper   M   Emily Browne
jumper   S   Tom Browne
jacket   M   Emily Browne
  • TIP: You can accumulate a string containing the list of items in the same way you accumulate the order total, in the loop used for reading the input.
