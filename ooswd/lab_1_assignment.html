Exercise 2: BMI calculator

Write a Python script that calculates the body-mass index. Specification:

  • Prompt the user for their name, then read the entered value and assign it to a suitable variable.
  • Prompt the user for their weight (in kg), then read in the entered value and assign it to a suitable variable.
  • Prompt the user for their height (in cm), then read in the entered value and assign it to a suitable variable.
  • Calculate the BMI (the weight in kilograms divided by the square of the height in meters), using the stored values for weight and height. Do not forget that the entered value is in centimeters.
  • Construct a report string from a template and specific information for the user. The result should be of the form "Thanks, Bob, your BMI is 25".
  • Print the report string to the console.
