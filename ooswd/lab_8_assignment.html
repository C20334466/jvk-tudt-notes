Lab assignment 8: File input and output

Assignment

  1. Using mockaroo.com create a csv file that has animal data (species, name, age, weight) for 200 animals.
  2. Define an animal class that holds all these data and a zoo class that holds a list of animal objects, both with a suitable constructor.
  3. Write a method in the zoo class that takes a file name as a parameter, then reads in the animal information from the file and adds all the animals to its list.
  4. Write a method in the zoo class called compile_transfer_list. This should randomly pick 10 animals that need to be transfered and write all their information into a file called transfer.csv in comma-separated-value (CSV) format.
  5. All the file input/output operations should be placed inside a try/except clauses.
  6. Import the numpy package.
  7. Write a method in the zoo class that converts weights from pounds to kilograms and the other way around, depending on a parameter. This method should map the list of animals to a list of weights and convert this to a numpy array before performing the conversion. Technically, this is quite unnecesary, but will give you a first taste for how numpy arrays work.
  8. Another method should calculate the average weight of the animals and print it out, also with the use of a numpy array.
  9. The classes should each be in a separate file and a third file called main.py should contain testing code which prints an easy-to-follow account of what is being tested and the results of the tests.
