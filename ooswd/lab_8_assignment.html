School of Computer Science

Lab assignment 8: File input and output

Assignment

  1. Using mockaroo.com create a csv file that has animal data (species, name, age, weight) for 200 animals.
  2. Define an animal class that holds all these data and a zoo class that holds a list of animal objects, both with a suitable constructor.
  3. Write a function in the zoo class that takes a file name as a parameter, then reads in the animal information from the file and adds all the animals to its list.
