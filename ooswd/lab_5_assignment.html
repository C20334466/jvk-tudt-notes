Week 5: Classes

Assignment

Your assignment is to implement a stock checking application with the use of Python classes. The requirements are as follows:

  • The application should consist of classes Product and Shop and a main program. Each class and the main program should be in a separate file (three files in all).
  • The Product class should have member variables for the product name, description and stock (string, string and integer respectively) and a constructor that takes parameters that populate each of the member variables.
  • The shop should hold a list of products and have functions for adding, removing and printing the list of products. It should also have a function that identifies the products that are low on stock (below 3) and prints a list of those products.
  • The file main.py should contain code for testing all the functionality of the Shop class. Start by loading the data below into a Shop object. 
    DATA = [
    ["milk", "Fresh from the farm", 10],
    ["bread", "Fresh from the baker", 4],
    ["tea", "Box of 100 teabags", 1],
    ["eggs", "A dozen per box", 33],
    ["ice cream", "1l box, vanilla", 2]
]

Submission

Create a zip file from the three Python files and submit the zip file in Brightspace using the upload link 'Lab 5'.
