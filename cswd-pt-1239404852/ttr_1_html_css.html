HTML and CSS Development Tools

  1. HTML and CSS development tools

    HTML and CSS development tools

    This is a list of some tools that can help with HTML and CSS development.

    Browser developer tools

    • HTML and CSS are special purpose declarative languages that rely on a special purpose interpreter (most often a browser) to do something with the declarations (most often to display something).
    • We can use browser developer tools to get more detailed and behind-the-scenes information about what the browser is doing, including about how it interprets the HTML and CSS.
    • In most browsers the developer tools are opened by pressing the F12 key.
    • HTML and CSS information is in a module of the tools called Inspector in Firefox and in one called Elements in Chrome. Here styles can be edited on the fly with temporary effect, for testing
    • Firefox has a module called Style Editor, which allows the editing and permanent update of currently open CSS files.

    HTML online validator

    The W3 Consortium provides this HTML validation tool: http://validator.w3.org.

  2. Tips and tricks

    Tips and tricks

    • Often it is easiest to center content by using the CSS declaration margin: auto;
    • Border settings can be specified using the shortcut CSS property border. So instead of:
      border-width: 5px;
border-style: solid;
border-color: black;
      we can use a single CSS declaration:
      border: 5px solid black;
    • More than one class can be specified for an element, e.g.: 
      <h1 class="big red">My title</h1>
