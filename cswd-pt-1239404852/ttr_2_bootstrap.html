Bootstrap

  • Bootstrap
    • is a front-end framework, providing HTML, CSS and Javascript libraries
    • is free!
    • was originally developed in Twitter
    • is now an open source project hosted on GitHub (since 2011)
  • What is Bootstrap for?
    • it provides various styling instructions for various html elements
    • it includes features for easy development of responsive websites
    • it includes mobile-first styles that can aid in development for mobile devices
  • How do Boostrap styles work?
    • Bootstrap library styles are associated with classes
    • to use a Bootstrap style you must
      • include the Bootstrap CSS library as a link in your html file (you can copy the link html element provided in section Introduction/Quick start/CSS on this web page)
      • make the element you want to style a member of the Bootstrap class associated with the desired style
