LAB: Setting up the CSWD Lab Git Repository

  1. Introduction

    Introduction

    Lab work contributes 10% to the final result in CSWD. To receive marks for a lab in any week of the term, the submitted work must be:

    • uploaded before the beginning of the following session
    • be in a folder (not zip file) named labN (where N is the lab number, e.g. lab1 or lab2)
    • contain evidence of a reasonable effort to complete the lab

    Uploads are to be made to your private repository in GitHub Classroom "CSWD-PT_2019-20S2". The section following this one, 'How to create the lab work repository' explains how this repository is created.

    Uploading ('pushing') to a GitHub repository is part of a wider version control system called git. Help on the required git commands can be found here.

  2. How to create the lab work repository (click me to find out)

    How to create the lab work repository

    1. In order to be able to access this module's GitHub Classroom, you must have a GitHub account. If you don't have an account with GitHub, the first thing you should do is to create an account at github.com.
    2. Once you have an account with GitHub, you can create a repository in the GitHub Classroom and prepare it for use. Complete the following steps to achieve this:
      1. Open the following URL in a browser: https://classroom.github.com/a/dqLeVi7W
      2. Log in if you are prompted (if you are already logged into GitHub in the browser, you will not be prompted)
      3. Click on the green button Accept this assignment when prompted
      4. The creation of your classroom repository will proceed automatically, at the end of which you should see this message:
      5. Click on the bottom link (after the text "Your assignment has been created:") to access your new repository
      6. Now, to initialise the repository you should add a README file to it by clicking README:
      7. The README file is opened and it contains some default text. If your GitHub account name does not identfy you, type your name into the README file to let the tutor know who you are. Then click on Commit new file to commit the README file to the repository
      8. Click on the link named cswd-pt-1920s2-labs-<your git name> to be brought to the main page of your newly created repository. You can copy or bookmark the URL of this repository, which is now displayed in the address field of your browser. This is the URL you will be using to clone the repository as explained in the git tutorial page.
