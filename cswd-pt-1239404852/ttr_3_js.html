Javascript Introduction

  1. Programming

    Programming with Javascript

    Programming

    • What is a computer program?

      A set of instructions that a computer knows how to carry out.

    • What is a programming language?

      Rules for how the program is specified (syntax and semantics).

    • What is a scripting language?

      A scripting language is generally a high-level interpreted language, used to automate tasks in some domain, e.g. operating system (e.g. bash in Linux), databases (e.g. SQL) or Web content (e.g. Javascript).

    Concepts

    • What are the two conceptual aspects of a program?

      Data (information) and algorithms (behaviour).

    • What concrete constructs correspond to the two conceptual aspects of a program?
      Data-representative constructs
      • literals
      • variables
      • types (primitive and classes)
      Algorithm-representative constructs
      • flow control (i.e. conditional statements and loops)
      • data manipulation (operators).

    Input/output

    • How does a computer program communicate with its environment (users, machines, other programs)?

      Through input and output in various forms.

    • What are a Web application's input and output with respect to the end-user?

      The input and output are contained in the Web user interface.

      Input
      anything that the user does (click, move mouse, reload page, scroll...)
      Output
      the appearance of the page, including media such as sounds and video

  3. Learning to Program (with Javascript)

    Learning to Program (with Javascript)

    • Learning to program is an infinite feedback loop involving understanding and practice: practice, practice, practice, work hard to understand, then again practice, practice, practice. Like with any language of any kind, it takes time and effort to acquire automatic understanding and become fluent.
    • Class work alone is not enough
    • The outcomes of successfully learning to program:
      • the knowledge of
        • a basic set of syntactic and semantic rules and how to apply them
        • what tools are available and how to use them
        • where to find required constructs and libraries and how to search reference material
      • the skill to
        • abstract real-world things into data and algorithms (design)
        • translate the abstractions into a programming language
      • the patience to
        • persist through many iterations of testing, debugging, correction and improvement of code
        • work through documentation and lack thereof to successfully integrate third-party elements into your code

  4. Tools for Developing in Javascript

    Tools for Developing in Javascript

    • The browser developer tools
      • Javascript console
        • To access the Javascript console in your browser, press key F12 to open the developer tools, then click the Console tab.
        • The Javascript interpreter inside the browser uses the Console to communicate (e.g. with information, warning and error messages).
        • You as the developer can use the console to interactively run Javascript code.
      • The debugger
        • To access the Javascript debugger in your browser, press key F12 to open the developer tools, then click the Debugger tab.
        • Use the Sources panel to navigate through the file system and open files.
        • Once a file is open, you can click on line numbers in the column to the left to set breakpoints.
    • Linting tools

      A linting tool checks the style, syntax and semantics of code. You will find an online Javascript linter here.
