Bootstrap 4: News Website with Mona Lisa

Assignment

  1. In VS Code, create an HTML file called index.html, containing the standard elements (html, head and body).
  2. In the head element of the file, add a link to the Bootstrap 4 CSS.
  3. By adding new HTML elements and Bootstrap 4 classes, create the navbar and layout shown in the picture. Do not add any HTML attributes or define any CSS rules yourself, with the following exceptions:
    • you may add class attributes in order to specify Bootstrap 4 classes
    • to the img elements you should add the src attribute with the correct value to include the given image
    • to the img elements you should add the height attribute with value 80 (height="80")
    • add an internal CSS rule with an id selector and use it to turn the second image upside down (the CSS rule is transform: rotate(180deg))
  4. Make sure that the Bootstrap 4 columns you use result in suitable responsive behaviour: as the width of the layout is reduced, the elements should be made to span the entire page before they start 'struggling' to accommodate their contents.
