Python and Tools

Python

  • A general-purpose programming language, Python is high-level and interpreted
  • Invented in the 1980s by Guido van Rossum in the Netherlands
  • It has specialised libraries for different applications, the most important ones for data analysis being Numpy, Matplotlib and Pandas

The tools

To work with Python, we will be using Jupyter Notebooks, which allow seamless combination of text and code, making them an apt tool for data analysis. Jupyter Notebooks can run online or locally.

Azure Notebooks
Jupyter Notebooks on Azure are a cloud-based implementation that you can log into using your TU Dublin microsoft account or a personal account. The credentials associated with your student account are:
  • username: <your student id i.e. x-number>@computing.stu.it-tallaght.ie
  • password: <your TU Dublin password>
Local installation on lab PCs
To run Jupyter Notebooks on a lab PC, open a command line window and navigate to the directory where your Jupyter Notebook files are. Then run the following command:

jupyter notebook
Local installation on your own computer
You may use Azure Notebooks or install Jupyter Notebook locally as explained here (the site has instructions on how to install JupyterLab - a more complex and sophisticated tool that includes Jupyter Notebook - or classic Jupyter Notebook).
