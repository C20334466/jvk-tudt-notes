Python

  • A general-purpose programming language, Python is high-level and interpreted
  • Invented in the 1980s by Guido van Rossum in the Netherlands
  • It has specialised libraries for different applications, the most important ones for data analysis being Numpy, Matplotlib and Pandas

The tools

To work with Python, we will be using Jupyter Notebooks, which allow seamless combination of text and code, making them an apt tool for data analysis.

Local installation on your own computer

There are a few alternatives:

  1. Anaconda, which can be downloaded from here and includes all the packages we need. [RECOMMENDED]
  2. Jupyter Labs provides the same tool as Anaconda but in standalone form. It can installed from here.
  3. Visual Studio Code extensions Python and Jupyter provide notebook functionality. More information can be found here.
Local installation on lab PCs

To run Jupyter Notebooks on a lab PC, open a command line window and navigate to the directory where your Jupyter Notebook files are. Then run the following command:

jupyter notebook
Online notebooks at Google Colab

These can be used with a Google account and allow you to work and save your files in the cloud (URL).
