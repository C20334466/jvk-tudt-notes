DataCamp Lessons

Lab work for this week is to complete three DataCamp chapters on Python, set as assignments in DataCamp classroom.

  1. If you have not already done so, sign up for DataCamp Classroom using the DataCamp classroom signup link in the Module Information section of the Moodle page and following the instructions provided with the link.
  2. [OPTIONAL] Complete the first three chapters of the course Introduction to Python (found under the Data Scientist with Python career). These chapters have not been set as DataCamp Classroom assignments as they involve topics covered in Software Development 1.
  3. Complete chapter NumPy of the course Introduction to Python (set as a DataCamp Classroom assignment)
  4. Complete chapter Matplotlib of the course Intermediate Python for Data Science (set as a DataCamp Classroom assignment)
  5. Complete chapter Dictionaries & Pandas of the course Intermediate Python for Data Science (set as a DataCamp Classroom assignment)
  6. Complete chapter Logic, Control Flow & Filtering of the course Intermediate Python for Data Science (set as a DataCamp Classroom assignment)
