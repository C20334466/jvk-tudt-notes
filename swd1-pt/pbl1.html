Problem-Based Learning Assignment: House Insulation Quotes

  • Your assignment is to write a Python program that automates the process of providing initial quotes to potential customers, for KeepWarm, a company that fits internal insulation.
  • You may use any development tools available on the lab PC but PyCharm is recommended (mostly for its debugging facility).
  • You may approach the task in any way you prefer, however, desiging the program on paper before you write any code is recommended.
  • Except where indicated, your program may assume that user input is going to be valid (i.e. error handling should be implemented only if stated as an explicit requirement).
  • Where user input is required, suitable prompts must be provided, e.g. Please enter your name:
  • Your solution must not use collections such as lists.
  • You should test your program and fix any problems before submitting.
  • The program should:
    • print a title banner e.g. *********** KeepWarm Quick Quotes *********** when it starts
    • read in the user's name and surname
    • read in the user's email address, check that the email address contains the @ symbol, then request a new one if not and repeat the process until an email address with the symbol is obtained
    • read in the height of the ceiling in the house for which the quote is being sought
    • read in the dimensions of a room, then ask the user if they wish to enter information for another room; repeat the process of reading in room dimensions until the user answers that they do not wish to enter information for another room
    • ask the user if they would be willing to have the work done during the summer months
    • calculate the estimated price for the internal insulation fitting, if it costs €50 per 1m2 of internal wall surface, including a €150 discount if the work can be done in the summer months
    • prepare a reference string, which should consist of
      • the first letter of the user's first name, capitalised
      • if the surname is 5 characters or longer, the first 5 letters of the user's surname, capitalised
      • if the surname is less than 5 characters long, the user's surname, capitalised and padded with the character X
      Examples:
      • name: Jane, surname: Sullivan, reference: JSULLI
      • name: Luke, surname: Kelly, reference: LKELLY
      • name: Bruce, surname: Lee, reference BLEEXX
    • print Your reference: <the prepared reference string>
    • print Your quote: €<the calculated estimated price>
