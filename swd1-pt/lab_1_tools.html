Installation of Tools for Software Development 1 and Introduction to their Use

  1. The tools and how to install them

    The tools and how to install them

    • Python
      • this is the interpreter for the language
      • it also includes essential libraries of code
      • when installed alone, it must be interacted with from the command line
      • it is installed in the labs
      • if working at home, you will need to install Python (the latest version) on your computer
      • Python is free to download and use
    • The integrated development environment (IDE) PyCharm
      • this is an environment that makes development with a programming language easier
      • it relies on Python to be installed in order to be in any way useful (it cannot interpret the language!)
      • is also installed on the lab computers
      • if working at home, you will have to install PyCharm (community edition)
      • the community edition is free

  2. Running Python code

    Running Python code

    Note: Sample commands are preceded by the characters '> ', representing a cursor. If the command is shown as 

    > python
    then all you have to type in at the command line is python.

    • Interactively from the command line (command-by-command)
      1. open a command line window on your computer (this will differ from OS to OS)
      2. type in
        
> python
      3. check that the cursor >>> is being displayed; this means that you are in the Python environment and can type Python code into the command line
      4. type in 
        
> 3 + 4
        and press Enter; what happens then?
      5. we will be trying out a few more commands in class...
    • As a script from the command line
      1. create and save a text file called lab1.py with the following line in it:
        3 + 4
      2. on the command line, change the current directory to the one containing the file lab.py
      3. type in
        python lab1.py
        what happens then?
      4. notice that this way of invoking Python code can be added to a script i.e. does not necessarily have to be called by typing in the line interactively
