HTML and CSS Tips

  1. Elements with more than one class

    Elements with more than one class

    More than one class can be specified for an element, e.g.: 
    <h1 class="spaced red">My title</h1>

    The following style and h1 elements in an HTML document:

    ...
    .spaced { letter-spacing: 1em; }
    .red { color: red; }
    ...
    <h1 class="spaced red">My title</h1>
    ...

    will be rendered as follows:

    My title

  2. Centering content

    Centering content

    Often it is easiest to center content by using the CSS declaration margin: auto;

  3. Shortcut for specifying border properties

    Shortcut for specifying border properties

    Border settings can be specified using the shortcut CSS property border. So instead of:

    border-width: 5px;
border-style: solid;
border-color: black;
    we can use a single CSS declaration:
    border: 5px solid black;
