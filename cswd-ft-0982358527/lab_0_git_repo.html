Uploading a first lab

  1. Setting up the CSWD Lab Git Repository

    Setting up the CSWD Lab Git Repository

    All lab work is to be uploaded to your private repository in GitHub Classroom CSWD-FT_2020-21-S1. To create this repository, do the following:

    1. Click on this GitHub Classroom invitation link.
    2. If you see the following page, log into GitHub or create an account if you do not have one.
    3. If you see the following page, click "Authorize github".
    4. On the following page, pick your X-number. If you do not have an X-number, your name should appear towards the top of the list of X-numbers (initial + surname) and you should pick that.
    5. When the pop-up shown below appears, check your number and click 'Ok'.
    6. On the next page click 'Accept this assignment'.
    7. Now you should see the page shown below. The given URL is the URL of your repository for CSWD labs.
    8. Click on the link to open your CSWD labs repository. Yous hould see the following page telling you how to initialise the repository. Click on 'README' (blue text).
    9. The following page allows you to edit the README file before saving. Type in your real name here if your GitHub id does not identify you. Click 'Commit new file'.
    10. The next page shows your initialised repository.

  2. Uploading the First Exercise

    Uploading the First Exercise

    1. Download and unzip the material for lab 'HTML and CSS Exercises' from the 'Client-Side Web Development' website.
    2. Complete Exercise 1. At the end of this you will have a file called Ex1.html.
    3. Log into your GitHub account and opn your CSWD lab repository.
    4. Upload the file Ex1.html from your computer, by clicking on 'Add file' and following the instructions.
